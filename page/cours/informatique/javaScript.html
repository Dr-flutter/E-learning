Cours de JavaScript

À propos de ce cours

Ce cours vous enseignera les bases de JavaScript, le langage de programmation qui rend le web interactif. Vous apprendrez à créer des programmes, à manipuler des données, et à ajouter de l'interactivité à vos pages web.

Durée estimée : 40 heures

Module 1 : Introduction à JavaScript

Découvrez les bases de JavaScript et son rôle dans le développement web.

Commencer

Module 2 : Variables et Types de Données

Apprenez à déclarer des variables et à travailler avec différents types de données.

Commencer

Module 3 : Structures de Contrôle

Maîtrisez les conditions et les boucles pour contrôler le flux de votre programme.

Commencer

Module 4 : Fonctions

Créez et utilisez des fonctions pour organiser et réutiliser votre code.

Commencer

Module 5 : Objets et Tableaux

Explorez les structures de données complexes en JavaScript.

Commencer

Module 6 : DOM et Événements

Apprenez à manipuler le DOM et à gérer les événements pour créer des pages web interactives.

Commencer

Projets pratiques

Mettez en pratique vos compétences JavaScript avec ces projets :

  • Créer une calculatrice interactive
  • Développer un jeu simple (par exemple, Pierre-Papier-Ciseaux)
  • Construire une application de liste de tâches
  • Créer une galerie d'images interactive
