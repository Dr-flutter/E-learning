Leçon 1 : Structure de base d'une page HTML

Introduction

Dans cette leçon, nous allons apprendre la structure de base d'une page HTML. HTML (HyperText Markup Language) est le langage de balisage standard utilisé pour créer des pages web. Comprendre sa structure fondamentale est essentiel pour tout développeur web.

La structure de base

Voici la structure de base d'une page HTML :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ma première page HTML</title>
</head>
<body>
    <h1>Bienvenue sur ma première page HTML</h1>
    <p>C'est excitant de créer ma première page web !</p>
</body>
</html>

Résultat :

Explication détaillée

La section <head>

La section <head> contient des informations importantes pour le navigateur :

  • <meta charset="UTF-8"> : Cela spécifie l'encodage des caractères pour le document (UTF-8 est recommandé pour sa compatibilité universelle).
  • <meta name="viewport" content="width=device-width, initial-scale=1.0"> : Cette balise est importante pour le design responsive, elle configure la vue sur les appareils mobiles.
  • <title> : Le titre qui apparaîtra dans l'onglet du navigateur.

La section <body>

C'est ici que vous placerez tout le contenu visible de votre page. Par exemple :

<body>
    <h1>Bienvenue sur ma première page HTML</h1>
    <p>Ceci est un paragraphe.</p>
</body>

Exercice pratique

Modifiez le code HTML ci-dessous pour ajouter une liste non ordonnée (ul) avec trois éléments (li) dans le corps de la page.

Résultat de l'exercice :

Quiz rapide

1. Quelle balise est utilisée pour définir le titre de la page qui apparaît dans l'onglet du navigateur ?

2. Où placez-vous le contenu principal visible de votre page web ?

Prochaine étape
